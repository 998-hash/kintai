# 勤務CSV集計ツール

Google Apps Script で勤務実績CSVを集計するためのスクリプトです。  
貼り付け用の本体コードは `payroll_aggregation_gas.txt` にあります。

## ファイル

- `payroll_aggregation_gas.txt`
  - Apps Script に貼り付ける集計コード

## 目的

勤務実績CSVを読み込み、`カレンダ名 × 勤務区分名` で単価マスタを参照して、
1か月分の勤務時間と金額を集計します。

## 対応シート

- `取込`
  - CSV原本を貼り付ける
- `単価マスタ`
  - `カレンダ名 / 勤務区分名 / 時給` を設定する
- `集計結果`
  - 行別計算結果
- `日別集約`
  - 1人1日の集計
- `月次集計`
  - 1か月分の合計

## 使い方

1. Google スプレッドシートを開く
2. Apps Script を開く
3. `payroll_aggregation_gas.txt` の内容を貼り付ける
4. `取込` シートに勤務CSVを貼り付ける
5. `単価マスタ` シートに単価を設定する
6. `runPayrollAggregation()` を実行する

## 単価マスタ

ヘッダは以下です。

- `カレンダ名`
- `勤務区分名`
- `時給`

`カレンダ名 × 勤務区分名` の組み合わせで時給を引きます。

## 集計ルール

- 8時間判定は1レコード単位ではなく、1人1日単位で行う
- 同じ個人コード・氏名・日付の行は同一日の勤務として扱う
- 同一日の通常支給時間は最大 8:00
- 8:00 を超えた分だけ時給 1.25 倍で計算する
- 同一日内の行配分は CSV 記載順で行う

行時間は以下で算出します。

- `所定時間 + 所定内残業 + 残業時間`

## このCSVへの対応

以下のCSVを基準に調整しています。

- `C:\Users\user01\Downloads\00000998_Xg0500_2603181905450\00000998_Xg0500_2603181905440.csv`

このCSVは `cp932` で、ヘッダに以下の特徴があります。

- `氏名` ヘッダが存在しない
- `個人コード` が2列ある
- 1列目の `個人コード` は社員コード
- 2列目の `個人コード` は氏名

スクリプトはこの形式を自動で解釈し、

- `個人コード` は1列目
- `氏名` は2列目の `個人コード`

として扱います。

通常の `氏名` ヘッダがあるCSVにも対応しています。

## エラー処理

- 単価未登録の行は `集計結果` の `エラー` 列に表示する
- 必須ヘッダが不足している場合は、検出したヘッダ一覧付きでエラーにする

## 備考

- 時間は `H:MM` 形式を分に変換して計算します
- `20260301` のような日付形式にも対応しています
- 出力金額は小数第2位まで保持します
